# 專業模型 (Professional Model) - 店面遷移公告網頁

這是一個靜態網頁專案，用於公告「專業模型」店面遷移的詳細資訊。網頁包含新店址資訊、多種交通方式指引（台鐵、捷運、開車）、停車建議以及聯絡方式。

## 🚀 專案功能

* **多種交通指引**：
    * 🚂 **台鐵**：民族車站步行路線。
    * 🚇 **捷運**：文化中心站步行路線。
    * 🅿️ **開車**：停車場導航與捷徑指引。
* **互動式地圖**：整合 Google Maps 連結，點擊即可開啟導航或街景。
* **一鍵聯絡**：提供直接撥打電話與加入 LINE 好友的功能。
